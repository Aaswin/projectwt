CSS rule set consist of a selector and a declarartion block.
SYNTAX :selector { declaration (property:vlaue);declaration }
SELECTOR :points to the html element we want to style.
The declaration block contains 1 or more declaration separated by semicolon.
Each declaration include a name and a value pair.
