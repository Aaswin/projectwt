FORM TAG

The form makes the web pages more interactive.It consists of form field.The user of the program fill up the field with the required information and then the information is then submitted to the web browser.There are generally six types of form tags used which should be written between the opening form tag i.e.<form> and the closing form tag i.e.</form>.The tags are:

A.Text Input Control Tag

It allows the user of the program to input the texts in the alloted area.It is of two types.They are single line input control tag and password input control tag.They both are empty tags.
Syntax for single line input control tag :<input type="text" name="xyz" value="pqr">
Syntax for password input control tag :<input type="password" name="xyz">

B.Check Box Control Tagg

More than one options can be selected using check box control tag.It can be created by using the input tag and the type attribute is set to checkbox.
Syntax :<input type="checkbox" name="xyz" value="pqr">

C.Radio Box Control Tag

It helps the user of the program to opt out anyone of the options among the given options.
Syntax :<input type="radio" name="xyz" value="pqr">

D.Select Box Control Tag

It is also called as a dropped down box which provides an option to list down the various options in the form of drop down list from where a user can select more than one options.
Syntax :<select name="dropdown"><option value="xyz"></option>....<option value="abc"></option>

E.Text Area Tag

The multiple lines of text are created using text area tag.It cannot be done by using input tag.It needs the attribute like rows,columns and name.
Syntax :<textarea name="Description" rows="5" cols="20"></textarea>

F.Submit and Reset Button

The submit button tag is used to create a button called submit which helps the user of the program to submit the informations filled up by him/her to the web browser.Similarly,the reset button tag creates a reset button which allows the user to reset the form to the initial stage by clearing all the informations filled up by him/her.
Syntax for Submit Button :<input type="submit">
Syntax for Reset Button :<input type="reset">
