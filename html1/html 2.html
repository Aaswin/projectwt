SOME BASIC TAGS





1.HTML Tag

It reperesents the the root of the html documents.It is a container that contain all the other html elements.
Syntax :<html></html>

2.HTML Tag

It represents the document header which can keep other html tags like title,links,etc.It contains a collection of meta data.
Syntax:<head></head>

3.HTML Tag

It represents the title name of the html document.The title element or the title contents are typically displayed in the browser's title bar.
Syntax :<title></title>

4.HTML Tag

It is used for declaring the main content section of the html document.The web page content is inserted between the opening and the closing tag.
Syntax :<body></body>

5.HTML Tag

It can use different size of a headings.It has six level of heading i.e.(in decreasing order of their sizes)h1,h2,h3,h4,h5 and h6.
Syntax :<h1></h1>,<h2></h2>...<h6></h6>

6.HTML Tag

Each paragraph of text should go between an opening tag <p> and closing tag </p>.
Syntax :<p></p>
