TABLE TAG

It helps to represent the informations or datas in tabular form.To create an efficient table row spans and column spans are used as an attribute.The process of merging one or more shells is known as span.If the shells are spanned rows wise then it is known as row span.If the shells are spanned column wise then it is known as column span.
Syntax :<table><tr><th></th>...<th></th></tr></table>

For example :
INPUT CODE :
<table border="1">
<tr>
<th rowspan="2">Name</th>
<th colspan="3">Mark</th>
</tr>
<tr>
<th>sub1</th>
<th>sub2</th>
<th>sub3</th>
</tr>
<tr>
<th>x</th><td>10</td><td>20</td><td>30</td>
</tr>
<tr>
<th>y</th><td>15</td><td>25</td><td>24</td>
</tr>
<tr>
<th>z</th><td>12</td><td>20</td><td>28</td>
</tr>
</table>
OUTPUT :
Name Mark
sub1 sub2 sub3
x102030
y152524
z122028
