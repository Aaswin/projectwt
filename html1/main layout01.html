HTML

First developed by Tim Berners-Lee in 1990, HTML is short for HyperText Markup Language. HTML is used to create electronic documents (called pages) that are displayed on the World Wide Web. Each page contains a series of connections to other pages called hyperlinks. Every web page you see on the Internet is written using one version of HTML code or another. HTML code ensures the proper for matting of text and images so that your Internet browser may display them as they are intended to look. Without HTML, a browser would not know how to display text as elements or load images or other elements. HTML also provides a basic structure of the page, upon which Cascading Style Sheets are overlaid to change its appearance. One could think of HTML as the bones (structure) of a web page, and CSS as its skin (appearance).

HTML TAGS

                                                                                                                                                                

HTML consists of a series of short codes typed into a text-file by the site author — these are the tags.The tags are what separate normal text from HTML code. You might know them as the words between the . They allow all the cool stuff like images and tables and stuff, just by telling your browser what to render on the page. Different tags will perform different functions. The tags themselves don’t appear when you view your page through a browser, but their effects do. The simplest tags do nothing more than apply formatting to some text.

HTML Attributes

Most of the html tags can also have attributes i.e.the extra bits of information.An attribute is used to define the characteristic of an html element and is placed inside the element's opening tag.All th eattributes are made up of two parts i.e.a name and value.
For example :<h1 style="color:red">
Here,style="color:red" is an attribute called style atrribute which assigns red color to the text in the above example.
