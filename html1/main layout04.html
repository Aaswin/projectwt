FRAME TAG




The frames are used to divide the browser into multiple sections where each section can lead an individual html document.The collection of frame is known as frame set.In order to use th eform on a page,we use frameset tag instead of body tag.The frameset tag defines how to divide the windows into frame.The row attribute of the frameset tag defines horizontal frame and the column attribute defines vertical frame.
Syntax :<frameset cols="30%,40%,30%"><frame src="a1.html"><frame src="a2.html">....<frame src="an.html"></frameset>
Here frame tags are the empty tags.
