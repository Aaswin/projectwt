LIST TAG

EXAMPLE

It is used to make list of the datas or the informations.It is divide into two types ordered list and unordered list.

1.ORDERED LIST

It assigns order serially to the list in the form of alphabet(lowercase or uppercase),numbers(decimal number or roman number) depending upon the programmers choice.
Syntax :<ol type="1"><li></li>...<li></li></ol>

2.UNORDERED LIST

It assign symbols(circle,disc,triangle,etc) to the list.
Syntax :<ul style="list-style-type:disc"><li></li>...<li></li></ul>

For example:
                                                         Ordered list:
                                                                                                                  INPUT CODE:
                                                                                                                  <html>
                                                                                                                  <head><title>List</title></head>
                                                                                                                  <body>
                                                                                                                  The list of vehicles are:
                                                                                                                  </br>
                                                                                                                  <ol type="1">
                                                                                                                  <li>Car</li>
                                                                                                                  <li>Bus</li>
                                                                                                                  <li>Bike</li>
                                                                                                                  </ol>
OUTPUT:
The list of vehicles are:
  1. Car
  2. Bus
  3. Bike

                                                         UnOrdered list:
                                                                                                                  INPUT CODE:
                                                                                                                  <html>
                                                                                                                  <head><title>List</title></head>
                                                                                                                  <body>
                                                                                                                  The list of vehicles are:
                                                                                                                  </br>
                                                                                                                  <ul style="list-style-type:disc">
                                                                                                                  <li>Car</li>
                                                                                                                  <li>Bus</li>
                                                                                                                  <li>Bike</li>
                                                                                                                  </ul>
OUTPUT: The list of vehicles are:
  • Car
  • Bus
  • Bike
