EXTERNAL CSS

an external css can be written in any text editor .it shouldnt contain any html tags. The style-sheet must be saved with '.css' extension. The external css url must be attributed to href attribute inside style inside head. external css once created can be applied multiple times. they help reduce the time to load the webpage and hence external css are prefered when using large css features.it is implemented by using <link>tag external css consists of three attributes: rel,type,href; syntax: eg:<link rel="stylesheet" type="text/css" href="pag1.css">











INLINE CSS

An inline CSS is used to apply a unique style to a single HTML element. An inline CSS uses the style attribute of an HTML element.It is repeated for evey element we want to put style to. Syntax:eg:<P style="font-size:30px ;color:red"> ->>>it will reder red color and size 30 px to content inside <p>tag.

















INTERNAL CSS

Internal css can be used to decorate a whole webpage with a single css style tag. it is put inside Html tags that such as <p>,<div> etc. Syntax:eg:<P style="font-size:30px ;color:red">
